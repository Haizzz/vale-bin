## About

NPM wrapper for [vale](https://github.com/errata-ai/vale "Link to vale on GitHub").

Downloads `vale` into a directory called *vendor* and makes it runnable.

## Install

```sh
npm install --save @ocular-d/vale-bin
```

## Usage

### CLI

```shell
vale-bin --help
```
